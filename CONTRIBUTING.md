# Contributing to Advance Clipboard Manager

Thank you for your interest in contributing! We welcome contributions from everyone.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/Advance-clipboard.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Install dependencies: `pip install -r requirements.txt`
5. Make your changes
6. Run the app to test: `python main.py`
7. Commit and push, then open a Pull Request

## Code Style

- Follow PEP 8 conventions
- Keep functions focused and small
- Add type hints where practical
- Match the existing code style (dark theme, Tkinter patterns)

## Pull Request Process

1. Ensure your code builds without errors
2. Update the README if needed
3. Include screenshots for UI changes
4. Reference any related issues

## Reporting Issues

Use the [issue tracker](https://github.com/yourusername/Advance-clipboard/issues) to report bugs or suggest features. Include:

- Python version and OS
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

## Code of Conduct

By participating, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).
